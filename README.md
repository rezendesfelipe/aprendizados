# Aprendizados (Learning Resources)

This repository is dedicated to providing a curated list of resources, people, and platforms that I recommend for studying various topics in technology.

## Categories

The resources are organized into the following categories:

- **[AI (Artificial Intelligence)](AI.md)** - Resources for learning about AI, machine learning, deep learning, and related topics
- **[Data Engineering](DataEngineering.md)** - Resources for learning about data engineering, data pipelines, ETL, and data infrastructure

## Structure

Each category file contains:
- **Resources** - Books, courses, tutorials, articles, and documentation
- **People** - Influential researchers, engineers, and educators to follow
- **Platforms** - Online platforms, tools, and communities for learning and practice

## Contributing

Feel free to suggest new resources by opening an issue or pull request!
